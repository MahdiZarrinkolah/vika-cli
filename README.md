# Vika Monorepo

This repository contains the entire Vika ecosystem:

- **cli/**: Rust-based API generator (vika-cli)
- **apps/docs/**: Documentation portal (Next.js, MDX) - coming soon
- **packages/playground/**: Code playground engine - coming soon
- **examples/**: Demo projects generated by vika-cli
- **tools/**: CI and deployment scripts

## Workspaces

This repository uses pnpm workspaces. The workspace configuration is defined in `pnpm-workspace.yaml` and includes:

- `apps/*` - Application packages
- `packages/*` - Shared library packages
- `examples/*` - Example projects

## CLI Documentation

For detailed documentation about the vika-cli tool, see [cli/README.md](cli/README.md).

The CLI is a blazing-fast Rust-powered OpenAPI code generator that produces TypeScript types, Zod schemas, and Fetch clients.

## Development

### CLI Development

```bash
cd cli
cargo build
cargo test
```

### Workspace Management

```bash
# Install dependencies (when packages are added)
pnpm install

# Run commands across workspaces
pnpm --filter <package-name> <command>
```

## Structure

```
vika-cli/
├── cli/                      # Rust CLI project
│   ├── src/                  # Source code
│   ├── builtin/              # Built-in templates
│   ├── docs/                 # CLI documentation
│   ├── schema/               # Configuration schemas
│   └── tests/                # Test suite
├── apps/
│   └── docs/                 # Documentation site (future)
├── packages/
│   └── playground/           # Playground engine (future)
├── examples/                 # Example projects
└── tools/                    # CI/CD scripts
```
